# Student Management System

A web-based Student Management System developed using Java, Spring Boot, Thymeleaf, and MySQL.

## Features

* Add Student
* Update Student
* Delete Student
* View All Students
* Search Students
* MySQL Database Integration

## Tech Stack

* Java 17
* Spring Boot
* Spring Data JPA
* Thymeleaf
* MySQL
* Maven

## Project Structure

* Controller Layer
* Service Layer
* Repository Layer
* Entity Layer
* MVC Architecture

## How to Run

1. Clone the repository
2. Configure MySQL in application.properties
3. Run the Spring Boot application
4. Open localhost:8080
